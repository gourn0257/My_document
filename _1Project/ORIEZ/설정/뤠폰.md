# 개요

축복과 저주의 세계안 [[그랜드 월드]] 사람들은 각각의 고유한 무기, **뤠폰**이 있다.
사람에 따라서 무기의 외형, 크기, 색깔,

# 순혈 뤠폰

```java
[ '책' ]
{
    인간: [ '검' ],
    엘파: [ '활', '단검', '화살' ],
    시어: [ '창' ],
    리우: [ '방패', '링' ],
    오크: [ '도끼' '사슬' ],
    드웨픈: [ '망치', '철구', '종', '악세사리?' ],
    몬: [ '스태프', '채찍', '구슬' ],
    얼리밋: [ '보석', '총', '포' ],
    : [ '' ],
    : [ '너클' ],
}

{
    정령: [ '낫', '갑주', '건틀릿' ],
}
```
